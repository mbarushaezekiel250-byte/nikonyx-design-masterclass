# Nikonyx Website Navigation

---

# Purpose

The navigation bar is the primary gateway to the Nikonyx ecosystem.

It must remain simple, elegant, and immediately understandable.

Users should always know:

- Where they are
- Where they can go
- How to return

# Sticky Navigation

The navigation bar should remain visible while scrolling.

Only reduce its height slightly.

Never disappear completely.

---

# Logo Rules

The Nikonyx logo always links back to the Home Page.

Never distort the logo.

Maintain consistent spacing.

---

# Navigation Behavior

Highlight the current page.

Hover states should be subtle.

Animations should last approximately 200–300 ms.

---

# Accessibility

Keyboard navigable.

Visible focus states.

High color contrast.

Readable font sizes.

---

# Design Rules

Use premium spacing.

Avoid overcrowding.

Keep navigation consistent across all pages.

---

# Future Expansion

The navigation should support additional products without redesigning the layout.

Examples:

- Marketplace
- Fashion Academy
- AI Studio
- Enterprise
