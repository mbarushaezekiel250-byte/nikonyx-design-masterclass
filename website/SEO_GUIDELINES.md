# Nikonyx SEO Guidelines

---

# Purpose

This document defines the Search Engine Optimization (SEO) standards for the Nikonyx website to improve discoverability, performance, and user engagement.

---

# SEO Principles

- Every page must have a unique title.
- Every page must have a unique meta description.
- Use descriptive and human-readable URLs.
- Prioritize fast page loading.
- Maintain semantic HTML structure.

---

# Page Titles

Each page should have a concise title of 50–60 characters.

Examples:

- Nikonyx | AI Personal Stylist
- Nikonyx | Virtual Try-On
- Nikonyx | Smart Wardrobe

---

# Meta Descriptions

Every page should include a description between 140–160 characters.

Descriptions should clearly explain the page content and encourage clicks.

---

# Heading Structure

Use only one H1 per page.

Follow a logical hierarchy:

- H1
- H2
- H3
- H4 (only when necessary)

---

# Images

Every image must include:

- Descriptive filename
- Alt text
- Optimized size
- Modern formats (WebP where appropriate)

---

# Performance

Target:

- Largest Contentful Paint (LCP) < 2.5 seconds
- First Input Delay (FID) < 100 ms
- Cumulative Layout Shift (CLS) < 0.1

Compress images and minimize JavaScript and CSS.

---

# Mobile SEO

The website must be mobile-friendly.

Responsive layouts are required.

Touch targets should be easy to use.

---

# Structured Data

Use structured data where appropriate for:

- Organization
- Products
- Articles
- FAQs

---

# Accessibility

Provide meaningful link text.

Use proper labels for forms.

Maintain sufficient color contrast.

Support keyboard navigation.

---

# Content Guidelines

Write original, valuable content.

Avoid duplicate content.

Use clear language focused on user needs.

---

# Success Metric

Every page should be optimized to achieve strong search visibility while delivering a fast, accessible, and high-quality user experience.
