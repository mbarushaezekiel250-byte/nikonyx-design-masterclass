# Nikonyx Responsive Rules

---

# Purpose

Nikonyx must provide a seamless experience across every device.

The interface should automatically adapt to different screen sizes while preserving usability, beauty, and performance.

---

# Supported Devices

## Desktop

Minimum Width:

1280px

Optimized for:

Large screens

Wide layouts

Professional workspaces

---

## Laptop

1024px+

---

## Tablet

768px+

Portrait

Landscape

Touch optimized

---

## Mobile

320px+

Responsive navigation

Large touch targets

Fast loading

---

# Layout Rules

Desktop

Multi-column layouts

Sidebar navigation

Large cards

---

Tablet

Reduce spacing

Collapse secondary panels

Adaptive grids

---

Mobile

Single-column layout

Bottom navigation

Hamburger menu

Stack cards vertically

---

# Responsive Images

Images must:

Scale automatically

Maintain aspect ratio

Load optimized versions

Support lazy loading

---

# Typography Scaling

Desktop

Large headings

Comfortable reading width

---

Tablet

Reduce heading sizes slightly

---

Mobile

Readable text

Minimum body size: 16px

---

# Touch Targets

Buttons

Minimum:

44px × 44px

Spacing between buttons should prevent accidental taps.

---

# Performance

Minimize layout shifts.

Optimize images.

Use lazy loading.

Reduce unnecessary animations on mobile.

---

# Accessibility

Support screen readers.

Keyboard navigation.

High color contrast.

Visible focus indicators.

---

# Testing

Every page should be tested on:

Desktop

Laptop

Tablet

Android

iPhone

---

# Success Metric

Users should enjoy a consistent experience regardless of device.
