# Nikonyx Animations

---

# Purpose

Animations should enhance the user experience without becoming distracting.

Every movement should feel smooth, elegant, and intentional.

---

# Animation Principles

Simple.

Natural.

Fast.

Meaningful.

Consistent.

# Performance

Animations should maintain 60 FPS whenever possible.

Avoid heavy animations on low-performance devices.

---

# Accessibility

Respect reduced motion preferences.

Provide functional alternatives when animations are disabled.

---

# Success Metric

Animations should make the interface feel premium while never slowing down the user's workflow.
