# Nikonyx Design Review Checklist

---

# Purpose

This checklist must be completed before any design is considered finished.

Every website page, mobile screen, dashboard, AI interface, and future Nikonyx product must pass every applicable check.

Design quality is not based on opinion.

It is based on measurable standards.

---

# Brand

□ Uses official Nikonyx colors

□ Uses approved typography

□ Matches Nikonyx visual identity

□ Looks premium

□ Looks modern

□ Looks clean

□ Feels consistent with previous screens

---

# Layout

□ Proper alignment

□ Proper spacing

□ Uses official grid

□ Balanced composition

□ No overcrowding

□ Clear hierarchy

---

# Typography

□ Readable

□ Consistent font sizes

□ Proper heading hierarchy

□ Comfortable line spacing

□ No unnecessary text

---

# Buttons

□ Correct size

□ Correct colors

□ Hover state exists

□ Disabled state exists

□ Loading state exists

□ Consistent spacing

---

# Forms

□ Labels present

□ Validation works

□ Error messages clear

□ Success messages clear

□ Keyboard accessible

---

# Navigation

□ Easy to understand

□ Current page highlighted

□ No dead links

□ Breadcrumbs where necessary

---

# Cards

□ Equal spacing

□ Proper shadows

□ Rounded corners

□ Consistent padding

---

# Images

□ High quality

□ Optimized

□ Proper cropping

□ Consistent style

□ No distortion

---

# Icons

□ Consistent style

□ Proper size

□ Proper spacing

# User Experience

---

# AI Review

Before approval ask:

Does this look like Nikonyx?

Would users trust it?

Is anything unnecessary?

Can anything be simplified?

Is the interface elegant?

Would Apple, Linear, Stripe, or Notion approve this level of polish?

---

# Final Approval

A design may only be approved when every required section above has been reviewed.

Quality is never optional.

Consistency is mandatory.
