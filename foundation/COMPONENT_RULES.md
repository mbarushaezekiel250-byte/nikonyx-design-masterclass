# Nikonyx Component Rules

---

# Purpose

This document defines the official UI component standards for every Nikonyx product.

Every website, mobile application, dashboard, AI assistant, and future digital product must use these rules.

Consistency is mandatory.

---

# Design Tokens

Every component must use the Nikonyx Design Tokens.

Never hardcode colors.

Never hardcode spacing.

Never hardcode font sizes.

Always reference the official design system.

---

# Button Rules

Buttons should never be smaller than 44px high.

Buttons should have consistent horizontal padding.

Button text should always be centered.

Never use more than one primary button inside a small dialog.
